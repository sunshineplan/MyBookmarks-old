My Bookmarks
======



Install
-------

Install MyBookmarks

    $ pip install -e .

Run
---

    $ export FLASK_APP=MyBookmarks
    $ export FLASK_ENV=development
    $ flask init-db
    $ flask run

Or on Windows cmd

    > set FLASK_APP=MyBookmarks
    > set FLASK_ENV=development
    > flask init-db
    > flask run

Open http://127.0.0.1:5000 in a browser.
